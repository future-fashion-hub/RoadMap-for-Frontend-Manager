# RoadMap-for-Frontend-Manager — дорожные карты и трекер прогресса

Проект RoadMap-for-Frontend-Manager — лёгкое приложение на React + TypeScript + Vite для просмотра и редактирования простых дорожных карт (roadmaps). Позволяет загружать JSON с описанием шагов, изменять статус, добавлять заметки и сохранять обновлённый файл.

**Ключевые возможности**
- Загрузка дорожной карты из JSON-файла
- Просмотр списка задач/пунктов в виде карточек
- Страница детали пункта с возможностью менять статус, дату и заметки
- Экспорт обновлённой дорожной карты в JSON

**Технический стек**
- React 19 + TypeScript
- Vite (dev server и сборка)
- React Router (страницы/роутинг)
- Lucide React (иконки)

Установка зависимостей и запуск

1. Установите зависимости:

```bash
npm install
```

2. Запуск в режиме разработки:

```bash
npm run dev
```

3. Сборка для продакшена:

```bash
npm run build
```

4. Просмотр собранного приложения (локально):

```bash
npm run preview
```

Как пользоваться

- На главной странице можно загрузить JSON-файл дорожной карты через контрол `Загрузить`.
- Каждый пункт отображается карточкой; клик по карточке открывает подробную страницу.
- На странице детали можно поменять статус (Не начат / В работе / Выполнено), установить дату завершения и добавить личные заметки.
- Экспорт сохранит текущую дорожную карту в файл `roadmap.json`.

Формат JSON

Пример минимальной структуры файла дорожной карты:

```json
{
	"name": "Frontend Roadmap",
	"description": "Основные навыки и темы для фронтенд-разработки",
	"items": [
		{
			"id": "1",
			"name": "HTML & CSS",
			"description": "Основы семантической верстки и стилей",
			"status": "completed"
		}
	]
}
```
